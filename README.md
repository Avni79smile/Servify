
# 🛠️ Servify

Servify is a comprehensive service marketplace and career portal built with React. It connects users with service professionals, tracks bookings, features an AI career assistant, and manages subscriptions with a robust Supabase backend.

## 🚀 Tech Stack

*   **Frontend:** React, JavaScript, Vite, Tailwind CSS, shadcn/ui
*   **Backend & Database:** Supabase (PostgreSQL, RLS Policies)
*   **Testing:** Vitest, Playwright

## 📦 Features

*   **Service Marketplace:** Browse, view professional profiles, and book services.
*   **Career Portal:** Job listings, application tracking, and an AI-powered career assistant.
*   **Dashboard Panels:** Dedicated views for Admin, Service Providers, and Customers.
*   **Real-time Features:** Chat messaging history and instant notifications.
*   **Secure Backend:** Row Level Security (RLS) policies protecting database transactions.

## 🛠️ Getting Started

Follow these steps to run the project locally:

1. **Clone the repository:**
```bash
   git clone [https://github.com/Avni79smile/Servify.git](https://github.com/Username/Servify.git)
   cd Servify

```

2. **Install dependencies:**

```bash
   npm install

```

3. **Set up environment variables:**
Create a `.env` file in the root directory and add your Supabase credentials:

```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

```

4. **Run the development server:**

```bash
   npm run dev

```
